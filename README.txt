# Mecha-Stark Biometric Login

A cool cyberpunk-style biometric login page with face recognition. It has animated mecha face, particle background, and real face detection using your webcam.

## What is this?

This is a futuristic login system that looks like something from a sci-fi movie. You can:
- Login with username and password (default: aryan / 1234)
- Login with your face using the webcam
- Register new faces (needs master code: 1234)

It has nice animations, shattering effects when switching to face mode, and glowing cyber elements.

## How to Use

1. Download the file `refined_mecha_stark.html`
2. Open it directly in your browser (Chrome or Edge recommended)
3. Allow camera permission when asked

### Login Options:

**Normal Login:**
- Username: `aryan`
- Password: `1234`
- Click "INITIALIZE"

**Face Login:**
- Click "LOGIN WITH FACE"
- Look at the camera
- Click "ANALYZE BIOMETRICS"

**Register New Face:**
- Click "Register New Face"
- Enter master code: `1234`
- Enter a name
- Click "ENROLL BIOMETRICS"

## Features

- Animated mecha face that assembles and shatters
- Interactive particle background that reacts to mouse
- Real face detection with face-api.js
- Stores faces in browser's IndexedDB
- Cool cyberpunk UI with neon effects
- Smooth transitions and animations

## Important Notes

- Works best on desktop with a webcam
- First time it may take a few seconds to load the AI models
- Face recognition is for fun/demo (not super secure)
- Make sure your face is well lit

Enjoy the futuristic login experience! ⚡
